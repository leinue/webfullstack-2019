Web 生来就是一个 通用 的平台

  • 每个人都可以自由的通过各种平台访问 Web：
    硬件
    PC
    手机
    笔记本电脑
    智能手表
    …
    软件
    操作系统
    浏览器
    移动 APP
    …
  • 开发者可以毫无限制的基于 Web 进行创新
    • 共建 Web 生态
    • 统一的 W3C 标准决定了 Web 的互操作性

Web 给予了所有人自由发声 的机会

  • 每个人都可以对任何事情做评论
  • 每个人都有自己的 Web 空间
    所以我们不需要去做任何妥协（Before 移动互联网）
  • 我们可以通过 超链接 访问无穷无尽的网站
    并在这些网站里发表评论和观点

互联网用户建立自己的网站和博客，
用他们喜欢的方式分享他们的知识、见解和故事

由此产生了很多社交平台连接用户
这些社交平台的初衷是帮助他们交流

Web 为全球创意工作者带来无门槛的创新体验

  • 每个人都可以通过 Web 将自己伟大的想法分发到全世界
  • Web 开发者不需要获得任何人的许可
    就可以随时随地发布自己的创意
    • 不是像 APP Stores 和小程序 那样需要审核

无门槛的创新为全球带来了前所未有的创意力量

同时我们的数据变得越来越中心化
—— 全部集中在巨头手中

  • 用户的个人博客
    现在在微信和微博上
    • 良好的用户体验
    • 但是我们失去了控制权
  • 越来越严重的隐私泄漏问题
  • 严重危害了 Web 的 通用性
    • 使用微信登录
    • 微信小程序

社交媒体之间高墙树立，
你无法自由移动数据和社交关系

当前巨大的难以撼动的中心化趋势
严重伤害了多元性、创新性和选择性。

  • 如果你只能创建一个 API 应用
    • 你会做 weixin.com 还是 identity-provider.org？
  • 开发者全部依靠 中心化平台 来管理数据和身份
    • 创业公司长大后受资本驱动也成为了中心化平台的一份子
  • 用户失去了自己对数据和身份的控制权
    同时也无法轻松的将数据在各个应用间转移
    • 创新逐渐闭塞，无法吸引更多有才华的人参与其中

每个人都可以搭建 SoLiD 服务器或使用 SoLiD 开发应用

  • 一台在智能音箱里的服务器
  • 现成的云服务
  • 免费的社区版本

SoLiD 服务器主要是作为数据 Pod
用来管理和守卫数据

  • 一个通常意义上的 HTTP Server
    • 支持 Access Control
    • 支持 Linked Data
    • 支持 OIDC
  • 通用应用程序开发平台
    • 构建 任意 应用
    • 将业务逻辑转移到前端
  • 就像你的个人网站一样
    • 你的数据可以被任意 APP 重复使用

一个典型的 Data Pod 可以存储任意数据

  • 个人资料 👤
  • 照片 🖼
  • 评论 🗣
  • 点赞 👍
  • 好友关系 👥
  • … ✨

SoLiD 客户端是浏览器应用或者移动原生应用
这些应用都从你自己的 Pod 中读取数据。

    • 可以进行非常细粒度的授权
  • 你的好友给你授予权限
    • 可以进行非常细粒度的授权
  • 应用提供统一的用户体验
    • 浏览你朋友的照片和你自己的照片

任何你能想到的应用，都可以使用 SoLiD 开发

PublicSpaces captures principles
to serve the common interest.

  • Open
  • Transparent
  • Accountable
  • Sovereign
  • User Centric

PublicSpaces principle: Open

  • The software ecosystem is equally accessible
    to any person, network or organisation.
  • The software ecosystem operates independently
    from any government, commercial
    or particular political influence.

PublicSpaces principle: Transparent

  • The used technology is fully accessible
    and reviewable.
  • Governance of the software ecosystem
    will be fully reviewable.

PublicSpaces principle: Accountable

  • Users are verified and provenance of content
    can be fully disclosed.
  • The ecosystem offers the highest possible guarantee
    of protection of data—both of content and users.
  • The software ecosystem does not hold or own
    any personal data of its users.
  • Governance of the software ecosystem is bound by
    the founding statutes that guarantee independence
    of commercial and political influences.

SoLiD principle: Sovereign

  • Users are not products—
    the wellbeing and empowerment of users
    will always be the primary concern.
  • Users have full control over
    their personal data,
    their content and interactions.

SoLiD principle: User Centric

  • The software ecosystem employs
    privacy by design
    as a basic design principle.
  • The software ecosystem uses the tenets of
    the Time well spent movement
    as guidelines for any implementation.

SoLiD 应用拥有多个后端，一个后端与多个应用进行交互。

SoLiD 使用 Linked Data 和 RDF 解决互操作性问题

  • 如果我们存储了你的数据，
    我们怎么和其他数据进行互联
  • 应用之间无摩擦的实现数据共享，
    在不需要改动代码的情况如何实现？
  • 如何在多个 Pod 之间对数据进行联合查询？

使用 JSON-LD，每片数据都可以和另一片数据进行互联
